# parallax-sample

parallax site sample

パララックスをcssで実装するサンプル


## setup

```sh
$ yarn

$ docker-compose up -d
```

## run fomatter

```sh
$ npm run fix
```

## Build scss

```sh
$ npm run build:sass
```

## view

```sh
$ open http://localhost:8080/
```
